## 📄 Descrição

Este repositório contém os diagramas UML modificados para a Avaliação NP2 da disciplina de Engenharia de Software, desenvolvida no contexto do PIMIII (Projeto Integrado Multidisciplinar III).

O projeto inclui uma análise completa do sistema de plataforma de ensino online (`PIM_III`) com representações visuais dos requisitos funcionais, estrutura de aulas e fluxos de interação.

## 📁 Conteúdo do Arquivo

O arquivo Astah (`JoaoLucas_PIMIII.asta`) contém os seguintes diagramas:

### 1. Diagrama de Casos de Uso

- Representa os atores do sistema
- Especificar os casos de uso principais
- Mostra os relacionamentos entre atores e funcionalidades
- Incluindo queda de uso para:
  - Acessar Comunicado
  - Incluir/Editar/Excluir Conteúdo
  - Gerenciar
  - Realizar custos
  - Acessar Relatórios
  - Gerenciar Usuários

### 2. Diagrama de Classes (Diagrama de Classes)

- Defina uma estrutura de classes do sistema
- Mostrar atributos e métodos
- Especificar as relações entre classes (associações, herança, agregações)
- Inclui aulas principais como:
  - Usuário
  - Aula
  - Conteúdo
  - Exercício
  - Avaliação
  - Comunicado
  - Relatório
  - Curso

### 3. Diagrama de Sequência

- Ilustra a interação entre objetos ao longo do tempo
- Mostra a ordem das mensagens trocadas
- Descreve o fluxo de execução dos casos de uso principais
- Cobre pontos críticos como:
  - Realização de touro
  - Processamento de rua
  - Publicação de comunicados
  - Acesso a conteúdo

Para visualizar e editar os diagramas, é necessário ter instalado:

## 📖 Como Usar

### Abrindo o arquivo no Astah:

1. Abra o Astah Professional ou Astah Community
1. Acesse: `File → Open → Selecione JoaoLucas_PIMIII.asta`
1. Navegue entre os diagramas usando a aba Modelo no lado esquerdo
1. Para visualizar cada diagrama, clique duas vezes em seu nome

### Exportando os Diagramas:

Para compartilhar em outros formatos:

```
# Exportar para XMI (XML)
Astah: File → Export → XMI

# Exportar para SVG (Imagem Vetorial)
Astah: File → Export → SVG

# Exportar para PNG/JPEG (Imagem)
Astah: File → Export → Image
```
